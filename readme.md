# API da padaria do seu João

## instalação

- instale as dependências a partir do requeriments.txt

```bash
pip install -r requeriments.txt
```

- Rode a aplicação utilizando o `uvicorn` 

## Rodando a aplicação

```bash
uvicorn main:app --reload
``` 

A aplicação a porta vai rodar na porta `8000`. Acesse `http://
localhost:8000` no seu navegador para ver a aplicação em ação.
